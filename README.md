# tucha 🎙

Терминальный голосовой мессенджер с TUI. Комнаты, микрофон, динамик, UDP аудио.

## Быстрый старт

### Скачать готовый бинарник

Зайди в [Releases](../../releases) и скачай версию под свою систему.

**macOS (M1/M2/M3):**
```bash
curl -L <ссылка на tucha-macos-arm64.tar.gz> | tar xz
chmod +x tucha
# При первом запуске macOS может заблокировать — см. ниже
./tucha
```

**Linux:**
```bash
curl -L <ссылка на tucha-linux-x86_64.tar.gz> | tar xz
chmod +x tucha
./tucha
```

**Windows:** скачай `.zip`, распакуй, запусти `tucha.exe`.

### macOS: обход Gatekeeper

Если macOS говорит «невозможно открыть» — это Gatekeeper блокирует неподписанный бинарник:
```bash
xattr -d com.apple.quarantine ./tucha
```
Или: правая кнопка мыши → «Открыть» → «Открыть».

## Подключение

При запуске появится экран подключения:
- **Адрес сервера** — `IP:7878` (спроси у того, кто поднял сервер)
- **Имя** — любой никнейм
- `Enter` — подключиться

## Клавиши

| Клавиша | Действие |
|---|---|
| `j` / `k` | навигация по списку |
| `Enter` | войти в комнату |
| `Tab` | переключить панель |
| `m` | включить/выключить микрофон |
| `d` | включить/выключить звук |
| `o` | выбор аудиоустройств |
| `q` | выйти |

## Запустить свой сервер

```bash
# Linux (Ubuntu/Debian)
sudo apt install libopus-dev
./tucha-server   # слушает TCP :7878 и UDP :7879
```

Открой порты в firewall:
```bash
sudo ufw allow 7878/tcp
sudo ufw allow 7879/udp
```
